WHAT IS ALTERNATIVE UNIVERSE IN DETAIL?

A web application that helps users to search for any space image fetched by the NASA Images Library API.

Alternative Universe is a space research project that enables researchers to download and search for space images directly from the huge NASA Images and Videos Library.

Alternative Universe is an educational, research, non-profit and non-commercial web application project
